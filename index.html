Minimalie

Minimalie is a custom ROM that is designed to be minimalist, fast, lightweight, and secure. It is based on the Android Open Source Project (AOSP), and it is free and open source software. Ready to build from our sources.

Minimalie comes with a number of features that are designed to improve and lighten the user experience, such as a custom launcher, and a number of modified open-source apps.

If you are looking for a fast, lightweight, secure, and minimal custom ROM, then Minimalie is the perfect choice for you.
